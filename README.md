# sberbank_test
Тестовое задание

Подключение к АПИ курсов биткойна не требует дополнительно авторизации или токенов, однако получение данных по АПИ гугл к сожалению возможно только через сервисный аккаунт. В связи с этим, считаю не безопасным делится своими доступами к кредам аккаунта, поэтому в коде есть участок кода получения данных, однако весь код расписан на основе обработки уже скачанного датафрейма.

Датафрейм транзакций расположен по ссылке https://disk.yandex.ru/d/_28yDEg02E4jhw

Это сделано в связи с ограничением github на размер загружаемых файлов

Предварительно при запуске тетрадки, требуется установка всех необходимых библиотек (при их отсутствии), командой pip install name_library
